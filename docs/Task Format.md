# Task Format
**Purpose**: The purpose of this document is to inform team members about how each of their tasks should be defined on the Trello board.

## Example
**Label & write tasks as follows**:
- Add a new feature (or Update the existing feature) "name" to "insert purpose of task here" by "explain how and by what methods this task will be accomplished".
- Note: each task should only have one deliverable. Otherwise break it into smaller tasks.
